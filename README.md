# js-obfuscator-for-rrp
JS-обфускация — это процесс преобразования исходного кода JavaScript в трудночитаемый формат с сохранением функциональности. Цель — защита интеллектуальной собственности, усложнение реверс-инжиниринга и сокрытие логики приложения. Методы включают переименование переменных, удаление комментариев, минификацию и запутывание потока управления.
